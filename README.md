# Introduction to Deep Learning Module, ESPCI Paris-PSL, Final Homework

This repository contains the final homework for the IDL module at ESPCI
Paris-PSL. 

The `DeepNovoV2_Notebook.ipynb` Jupyter notebook is a re-implementation of the
DeepNovoV2 model by [Qiao _et al._](https://arxiv.org/abs/1904.08514). The
`smbp_deepnovov2_tools.py` Python module contains helper functions to
facilitate the generation of input files for DeepNovoV2, from any MGF file and
the associated Mascot XML result files. The `generate_input_deepnovov2.ipynb`
Jupyter notebook demonstrates the use of the module.
